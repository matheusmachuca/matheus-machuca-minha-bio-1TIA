# Minha Biografia

Este é um projeto simples de biografia pessoal, criado com HTML e CSS.

## Estrutura do Projeto

- `index.html`: Página principal com informações pessoais e interesses.
- `interesses.html`: Página detalhada dos interesses pessoais.
- `css/style.css`: Arquivo de estilos CSS para o layout e design.

## Como Visualizar

Abra o arquivo `index.html` em um navegador web para visualizar a biografia.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Organização

O projeto foi organizado com:
- Estrutura semântica usando `<header>`, `<main>`, `<section>` e `<footer>`.
- Estilos centralizados no arquivo `css/style.css`.
- Layout responsivo com container centralizado.
